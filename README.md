## Adding wallpapers

Drop files into the correct folder, name them `<palette>-<subject>-<nn>.png`:

```
pixel-art/navy-city-01.png
pixel-art/teal-forest-02.png
```

Palette prefixes: `navy`, `teal`, `rose`, `amber`, `mono`, `multi`
